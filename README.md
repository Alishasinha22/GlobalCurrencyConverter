# Global Currency Converter

The Global Currency Converter is a responsive web application that allows users to convert currencies in real time using live exchange rate data. The application supports over 40 international currencies and provides a clean, intuitive interface designed for usability and clarity.

This project demonstrates practical frontend development skills, API integration, and real-world data handling using modern JavaScript.


## Live Demo
https://alishasinha22.github.io/GlobalCurrencyConverter/

## Project Overview
The application fetches real-time exchange rates and allows users to convert an entered amount between two selected currencies. It also displays the current exchange rate, maintains a conversion history, and offers quick-conversion shortcuts for commonly used currency pairs.

The design focuses on simplicity, responsiveness, and user experience while maintaining accurate and up-to-date financial data.


## Features
- Real-time currency conversion  
- Support for 40+ global currencies  
- Clean and responsive user interface  
- Light and dark theme support  
- Quick conversion buttons for popular currency pairs  
- Conversion history tracking  
- Manual refresh option for exchange rates  
- Displays last updated timestamp for exchange rates  


## Technologies Used
- **HTML5** – Semantic structure and layout  
- **CSS3** – Styling, layout, and theme management  
- **JavaScript (ES6+)** – Application logic and API integration  
- **Frankfurter API** – Live exchange rate data based on ECB reference rates

  
## API Details
This project uses the **Frankfurter Exchange Rates API**, a free and reliable exchange rate service.
- Exchange rates are updated every 60 minutes  
- Rates are based on European Central Bank (ECB) data  
- Uses mid-market exchange rates  
- No API key or authentication required  

## Supported Currencies
The application includes major international currencies such as:
USD, EUR, GBP, INR, JPY, CAD, AUD, CHF, CNY, SGD, KRW, MXN, BRL, ZAR, AED, SAR, THB, TRY, IDR, MYR, PHP, VND, HKD, BDT, LKR, NPR, and more.
